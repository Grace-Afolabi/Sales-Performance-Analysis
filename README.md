## **Project Overview**

This project presents a comprehensive **Sales Performance Dashboard** for **Adidas**, built to visualize key business metrics and uncover performance insights across regions, products, retailers, profit margins and price trends.
The dashboard enables quick decision-making by highlighting sales drivers, profit trends, and top-performing channels.
